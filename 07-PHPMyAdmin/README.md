# Aula 07 - PHPMyAdmin (Parte 1)

## 🎯 Objetivo

Aprender como acessar e manipular um banco de dados MySQL utilizando diferentes ferramentas, como MySQL Workbench, Console MySQL e phpMyAdmin.

## 📚 Comandos utilizados

- SHOW DATABASES
- STATUS
- USE 
- SHOW TABLES
- DESCRIBE
- SELECT * FROM 
- UPDATE 
- SET
- WHERE 

✅ Resultado
Acessei o banco de dados pelo Console MySQL utilizando comandos SQL.
Naveguei entre bancos de dados e tabelas sem depender de uma interface gráfica.
Utilizei o phpMyAdmin para visualizar e editar registros de forma gráfica.
Reforcei o uso dos comandos SQL já estudados anteriormente em diferentes ambientes.

📚 O que eu aprendi

- O comando `SHOW DATABASES`; lista todos os bancos de dados existentes no servidor.
- O comando `STATUS`; exibe informações da conexão atual, incluindo o banco de dados em uso.
- O comando `USE` seleciona o banco de dados que será utilizado durante a sessão.
- O comando `SHOW TABLES`; lista todas as tabelas do banco selecionado.
- Os comandos `DESCRIBE` e `DESC` exibem a estrutura de uma tabela.
- O comando `SELECT` permite consultar os registros armazenados nas tabelas.
- O comando `UPDATE` altera dados existentes, e essas alterações podem ser visualizadas imediatamente em qualquer ferramenta conectada ao mesmo servidor.
- O phpMyAdmin gera automaticamente os comandos SQL correspondentes às ações realizadas pela interface gráfica, tornando-se uma excelente ferramenta para aprender SQL.
- Aprendi que conhecer os comandos SQL é essencial para utilizá-los futuramente em linguagens como PHP, Java e outras que acessam bancos de dados.
- Aprendi que qualquer alteração realizada no Console, Workbench ou phpMyAdmin é refletida imediatamente nas demais ferramentas, pois todas acessam o mesmo servidor MySQL.
