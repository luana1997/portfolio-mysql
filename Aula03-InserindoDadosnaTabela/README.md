Aula 03 - Inserindo Dados na Tabela (INSERT INTO)

🎯 Objetivo

Aprender a inserir registros em uma tabela utilizando o comando INSERT INTO, adicionando informações de forma manual e compreendendo as boas práticas para o preenchimento dos dados.

📚 Comandos utilizados
- INSERT INTO
- VALUES
- Inserção de um único registro
- Inserção de vários registros em um único comando
