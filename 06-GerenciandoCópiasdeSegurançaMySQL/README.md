# Aula 06 - Gerenciando Cópias de Segurança MySQL

## 🎯 Objetivo
Aprender a criar um backup (dump) de um banco de dados MySQL, localizar o arquivo gerado, excluir um banco de dados e restaurá-lo a partir do backup.

## ✅ Resultados

- Criei um backup completo de um banco MySQL.
- Gerei um arquivo .sql contendo toda a estrutura e os dados do banco.
- Localizei o arquivo de backup no computador.
- Exclui um banco de dados com segurança.
- Restaurei completamente um banco de dados utilizando um arquivo de dump.

## 📖 O que aprendi

- Entendi a importância de realizar backups periódicos de bancos de dados.
- Aprendi que o mysqldump é a ferramenta utilizada para exportar toda a estrutura e os dados de um banco MySQL.
- Aprendi onde localizar o arquivo de backup (dump) salvo no computador.
- Aprendi a restaurar um banco de dados utilizando um arquivo de backup criado anteriormente.
- Entendi que a restauração recria tabelas, registros, índices e demais objetos existentes no momento do backup.
- Entendi como funciona um processo de recuperação de dados.
